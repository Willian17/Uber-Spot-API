# Uber-Spot-API


### Criar um spot (ponto virtual)

POST https://uber-spot.herokuapp.com/spot

#### request.body: 

    {
      "latitude": -16.4446755,
      "longitude": -54.6981687
    }
    
#### response da request: 

    {
      "_id": "5fb0587cbe86941d03bb4c7f",
      "coordinates": [
        -16.4446755,
        -54.6981687
      ],
      "__v": 0
    }
